# TC-009

| Field | Description |
|---|---|
| **Test ID** | TC-009 |
| **Feature** | Mood Tracker |
| **Scenario** | Edge case — Submit mood log with empty required field (should fail gracefully) |
| **Steps** | 1. Log in with a valid account. 2. Navigate to the Mood Tracker page. 3. Do not select a mood score — leave the required field empty. 4. Attempt to submit the form. |
| **Expected Result** | The form does not submit. A validation message is shown. No record is written to `mood_logs`. |
| **Actual Result** | Form correctly blocked submission and displayed a validation error. No null or invalid entry was written to the database. |
| **Status** | Pass |
| **Notes** | Confirmed no record inserted into `mood_logs` table. Validation handled on the front end. |
